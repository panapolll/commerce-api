# Screenshots

Add these images to this folder, then they will appear in the README:

| File | Page to capture |
|------|-----------------|
| `login.png` | Login page |
| `products.png` | Products listing |
| `cart.png` | Shopping cart |
| `payment.png` | Payment page |
| `orders.png` | Order history |

**How to capture:** Run `yarn dev`, open each page, press `Win + Shift + S` (Windows) or screenshot tool, save here.
